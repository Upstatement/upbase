    clearfix

    @mixin clearfix() { ... }

    Description

    Force an element to self-clear its children.

    Parameters

    None.

    Example

    HTML

    <section class="container">
    <div class="col-1 col">
        <!-- Content Goes Here -->
    </div> <!-- /col-1 -->
    <div class="col-2 col">
        <!-- Content Goes Here -->
    </div> <!-- /col-2 -->
</section> <!-- /container -->

    SCSS

    .container {
  @include clearfix();
  .col-1,
  .col-2 {
    float: left;
    width: 50%;
  }
}

    CSS Output

    .container:after {
  content: "";
  display: table;
  clear: both;
}

    Used by

    • [mixin] media-grid

    svg

    @mixin svg($bg-name, $bg-position: 0 0, $bg-color: transparent, $bg-size: auto) { ... }

    Description

    Use SVGs as background images with the proper PNG fallbacks. You'll need to set the display (inline-block or block) and a height/width for the container. Also, before using, set a default image path if you wish by overriding the empty $default-img-path variable in your project's settings or variables file.

    For full support, produce your graphic with highest resolution in mind, and export:

    • A responsive-ready SVG named {file-name}.svg
    • A highest-resolution PNG named {file-name}.png
    • A 1x-resolution PNG named {file-name}-ie.png

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $bg-name

    Name of the image file without extension. Include a path if you have not set a default $img-path variable or if you want to use a unique path.

    		String—none
    $bg-position

    background-position

    		Number[unit]0 0
    $bg-color

    background-color

    		Stringtransparent
    $bg-size

    background size

    		Stringauto

    Example

    SCSS

    //In your project's settings or variables file
$default-img-path: "http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/";

.basic {
  @include svg(envelope);
}

.awesome {
  @include svg(envelope, 50% 50%, #bada55, 36px auto);
}

.moreawesome {
  @include svg(envelope, 50% 50%, #bada55, cover);
}

    CSS Output

    .moreawesome {
  background-image: url("http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/envelope.png");
  background-image: url("http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/envelope.svg"), none;
  background-color: #bada55;
  background-repeat: no-repeat;
  background-position: 50% 50%;
  background-size: cover;
}
.no-svg .moreawesome {
  background-image: url("http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/envelope.png");
}
.ie8 .moreawesome {
  background-image: url("http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/envelope-ie.png");
}

    Requires

    • [variable] default-img-path
    • [variable] support-ie8
    • [variable] using-modernizr

    Links

    • http://codepen.io/upstatement/pen/qdpOdw
    • http://css-tricks.com/svg-fallbacks/

    vertical-center

    @mixin vertical-center() { ... }

    Description

    Vertically center the content of a container, regardless of the content height.

    Parameters

    None.

    Example

    HTML

    <div class="ui-block">
  <div class="ui-block-inner">
    <!-- Content Goes Here -->
  </div><!-- /ui-block-inner -->
</div><!-- /ui-block -->

<div class="ui-block">
  <div class="ui-block-inner">
    <!-- Content Goes Here -->
  </div><!-- /ui-block-inner -->
</div><!-- /ui-block -->

<div class="ui-block">
  <div class="ui-block-inner">
    <!-- Content Goes Here -->
  </div><!-- /ui-block-inner -->
</div><!-- /ui-block -->

    SCSS

    .ui-block-inner {
  @include vertical-center;
}

    CSS Output

    .ui-block-inner {
  position: relative;
  top: 50%;
  transform: translateY(-50%);
}
.lt-ie9 .ui-block-inner {
  top: auto;
}

    Requires

    • [variable] support-ie8
    • [variable] support-ie7

    Links

    • http://codepen.io/upstatement/pen/NqXGdv

    grid-static

    @mixin grid-static($width, $margin, $static-col: left) { ... }

    Description

    A two-column layout with one fixed-width column and one liquid-width column.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $width

    Width of the static column

    		Number [unit]—none
    $margin

    Margin between the two columns

    		Number [unit]—none
    $static-col

    Choose the left or right column as the static column

    		Stringleft

    Example

    HTML

    <div class="l-static">
  <section class="ui-block-1">
    <div class="ui-block-inner">
      <!-- Block 1 Content Goes Here -->
    </div> <!-- /ui-block-inner -->
  </section> <!-- /ui-block-1 -->
  <section class="ui-block-2">
    <div class="ui-block-inner">
      <!-- Block 2 Content Goes Here -->
    </div> <!-- /ui-block-inner -->
  </section> <!-- /ui-block-2 -->
</div> <!-- /l-static -->

    SCSS

    .l-static {
  @include grid-static(300px, 40px);
}

    CSS Output

    .element {
  padding: 30px -20px 10px 2.35765%;
}

    media-grid

    @mixin media-grid($num: 2, $pad: 20px, $target: ".media-block") { ... }

    Description

    Used to create a grid of items with equal widths. Great for simple 50/50, 33/33/33, etc. grids. Particularly useful when designing for an unknown or varying number of items.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $num

    Number of items per row

    		Number2
    $pad

    Padding between items

    		Number[unit]20px
    $target

    Class name of target items

    		String".media-block"

    Example

    HTML

    <section class="grid-3">
    <div class="grid-item">
        <!-- Content Goes Here -->
    </div> <!-- /media-block -->
    <div class="grid-item">
        <!-- Content Goes Here -->
    </div> <!-- /grid-item -->
    <div class="grid-item">
        <!-- Content Goes Here -->
    </div> <!-- /grid-item -->
</section> <!-- /three-up -->

    SCSS

    .grid-3 {
  @include media-grid(3, 10px);
}

.grid-4 {
  @include media-grid(4, 10px);
}

.grid-5 {
  @include media-grid(5, 10px, ".grid-item");
}

    CSS Output

    .grid-3 {
  margin-left: -10px;
}
.grid-3:after {
  content: "";
  display: table;
  clear: both;
}
.grid-3 > .media-block {
  float: left;
  padding-left: 10px;
  width: 33.33333%;
}
.ie7 .grid-3 > .media-block {
  width: 33%;
}
.grid-3 > .media-block:nth-of-type(3n+4) {
  clear: left;
}

    Requires

    • [mixin] clearfix
