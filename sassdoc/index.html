    em

    @function em($px-val, $em-base) { ... }

    Description

    Convert pixels to ems. $em-base is set globally to 16px in Upbase's settings file. You can override it in your project's settings or variables file.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $px-val

    Desired pixel equivalent

    		Number—none
    $em-base

    Parent em-to-pixel value

    		Number—none

    Example

    SCSS

    font-size: em(12);
font-size: em(12, 24);

    CSS Output

    font-size: 0.75em;
font-size: 0.5em;

    Requires

    • [variable] em-base
    • [function] strip-units

    Links

    • CSS Tricks: rems / ems

    hide-text

    @mixin hide-text() { ... }

    Description

    Hides text, but shows everything else like background colors, images, etc.

    Warning: This has been known to mistakenly fail as invalid CSS, and apparently it fails completely in Opera Mini, with the text fallback completely obscured. Recommendation is to use proper mixins, with accessibility considered, such as visually-hidden, hidden, and invisible despite them requiring two separate elements.

    Parameters

    None.

    Example

    HTML

    <div class="item">Lorem ipsum dolor sit amet.</div>

    SCSS

    .item {
  @include hide-text;
  &:after {
    //This :after element will still appear.
    content: " ";
    display: block;
    height: 16px;
    width: 16px;
    background-color: black;
  }
}

    CSS Output

    .item {
  color: transparent;
  font: 0/0 a;
  text-shadow: none;
}

    Links

    • https://css-tricks.com/examples/ImageReplacement/
    • http://codepen.io/upstatement/pen/aOYyEo

    svg

    @mixin svg($bg-name, $bg-position: 0 0, $bg-color: transparent, $bg-size: auto) { ... }

    Description

    Use SVGs as background images with the proper PNG fallbacks. You'll need to set the display (inline-block or block) and a height/width for the container. Also, before using, set a default image path if you wish by overriding the empty $default-img-path variable in your project's settings or variables file.

    For full support, produce your graphic with highest resolution in mind, and export:

    • A responsive-ready SVG named {file-name}.svg
    • A highest-resolution PNG named {file-name}.png
    • A 1x-resolution PNG named {file-name}-ie.png

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $bg-name

    Name of the image file without extension. Include a path if you have not set a default $img-path variable or if you want to use a unique path.

    		String—none
    $bg-position

    background-position

    		Number[unit]0 0
    $bg-color

    background-color

    		Stringtransparent
    $bg-size

    background size

    		Stringauto

    Example

    SCSS

    //In your project's settings or variables file
$default-img-path: "http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/";

.basic {
  @include svg(envelope);
}

.awesome {
  @include svg(envelope, 50% 50%, #bada55, 36px auto);
}

.moreawesome {
  @include svg(envelope, 50% 50%, #bada55, cover);
}

    CSS Output

    .moreawesome {
  background-image: url("http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/envelope.png");
  background-image: url("http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/envelope.svg"), none;
  background-color: #bada55;
  background-repeat: no-repeat;
  background-position: 50% 50%;
  background-size: cover;
}
.no-svg .moreawesome {
  background-image: url("http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/envelope.png");
}
.ie8 .moreawesome {
  background-image: url("http://www.thetrace.org/wp-content/themes/thetrace/static/img/icons/envelope-ie.png");
}
